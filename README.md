# 🚀 Mfa Chat - নিওন যুগের চ্যাটিং

**Mfa Chat** একটি পূর্ণাঙ্গ রিয়েল-টাইম চ্যাট অ্যাপ্লিকেশন, যা Facebook Messenger-এর মতো ফিচার সমৃদ্ধ। নিওন-স্টাইল ডিজাইন, রিয়েল-টাইম মেসেজিং, গ্রুপ চ্যাট, ফ্রেন্ড ম্যানেজমেন্ট এবং PWA সাপোর্ট সহ সম্পূর্ণ ফ্রি ও ওপেন সোর্স।

---

## ✨ ফিচারসমূহ

| ফিচার | বিবরণ |
|--------|--------|
| 🔐 **অথেন্টিকেশন** | JWT দিয়ে নিরাপদ রেজিস্টার ও লগইন |
| 💬 **রিয়েল-টাইম চ্যাট** | Socket.io দিয়ে ইনস্ট্যান্ট মেসেজিং |
| 👥 **গ্রুপ চ্যাট** | একাধিক বন্ধু নিয়ে গ্রুপ তৈরি ও চ্যাট |
| 👤 **ফ্রেন্ড ম্যানেজমেন্ট** | বন্ধু খোঁজা, অ্যাড, অ্যাকসেপ্ট, রিমুভ |
| 🟢 **অনলাইন/অফলাইন** | রিয়েল-টাইম ইউজার স্ট্যাটাস |
| 😊 **ইমোজি পিকার** | মেসেজে ইমোজি পাঠানো |
| ❤️ **মেসেজ রিয়েক্ট** | মেসেজে হার্ট রিয়েক্ট |
| 🖼️ **প্রোফাইল আপলোড** | Cloudinary দিয়ে প্রোফাইল ছবি |
| 🔔 **নোটিফিকেশন** | ডেস্কটপ ব্রাউজার নোটিফিকেশন |
| 📱 **PWA সাপোর্ট** | মোবাইলে অ্যাপের মতো ইনস্টল |
| 🛡️ **অ্যাডমিন প্যানেল** | ইউজার ও ড্যাশবোর্ড ম্যানেজমেন্ট |
| 📖 **ব্যবহার নির্দেশিকা** | বাংলায় সম্পূর্ণ গাইড |
| 🎨 **WhatsApp স্টাইল UI** | পরিচিত ও সুন্দর ডিজাইন |

## 🛠️ টেকনোলজি স্ট্যাক

### ফ্রন্টএন্ড
- ⚛️ **React** (Vite)
- 🎨 **Tailwind CSS**
- 🔄 **Socket.io Client**
- 📡 **Axios**
- 😊 **Emoji Picker React**

### ব্যাকএন্ড
- 🟢 **Node.js** + **Express.js**
- 🔌 **Socket.io**
- 🔐 **JWT + bcrypt.js**
- 🗄️ **MongoDB** (Mongoose)
- ☁️ **Cloudinary**

### ডেপ্লয়মেন্ট
- ▲ **Render** (ফ্রন্টএন্ড + ব্যাকএন্ড)
- 🗄️ **MongoDB Atlas**
- ☁️ **Cloudinary**

## 📦 ইনস্টলেশন (লোকাল)

```bash
# রিপোজিটরি ক্লোন
git clone https://github.com/Fuyad2751/Nion-Mfa-Chat-.git
cd Nion-Mfa-Chat-

# ব্যাকএন্ড সেটআপ
cd server
npm install
# .env ফাইল তৈরি করে নিচের তথ্য দাও:
# MONGO_URI, JWT_SECRET, PORT, CLOUDINARY_*

# ব্যাকএন্ড রান
npm run dev

# ফ্রন্টএন্ড সেটআপ (নতুন টার্মিনাল)
cd client
npm install
npm run dev
